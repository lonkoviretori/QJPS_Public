## 2.1 内积空间的概念

### 1. 核心定义
*   **定义 2-1（内积 - 实数域）**：设 $V$ 是实数域 $\mathbb{R}$ 上的线性空间，对任意 $\alpha, \beta \in V$，定义一个实数 $(\alpha, \beta)$，若满足：
    1.  **对称性**：$(\alpha, \beta) = (\beta, \alpha)$
    2.  **齐次性**：$(k\alpha, \beta) = k(\alpha, \beta)$
    3.  **可加性**：$(\alpha + \beta, \gamma) = (\alpha, \gamma) + (\beta, \gamma)$
    4.  **正定性**：$(\alpha, \alpha) \ge 0$，且 $(\alpha, \alpha) = 0 \iff \alpha = \mathbf{0}$
    称其为内积，定义了内积的实线性空间称为**欧氏空间 (Euclidean Space)**。
*   **定义 2-2（长度/模）**：$|\alpha| = \sqrt{(\alpha, \alpha)}$。
*   **定义 2-3（夹角与正交）**：
    *   $\cos \varphi = \frac{(\alpha, \beta)}{|\alpha| |\beta|}$（仅限实空间）。
    *   若 $(\alpha, \beta) = 0$，称 $\alpha, \beta$ **正交**（垂直），记作 $\alpha \perp \beta$。

### 2. 例题引申的新概念与结论
*   **标准内积模型**：
    *   **$\mathbb{R}^n$ 空间**：$(\alpha, \beta) = \sum x_i y_i = Y^T X$。
    *   **矩阵空间 $\mathbb{R}^{m \times n}$**：$(A, B) = \sum a_{ij}b_{ij} = \text{tr}(B^T A)$。
    *   **连续函数空间 $C[a,b]$**：$(f, g) = \int_a^b f(x)g(x) dx$。
*   **单位向量**：模长为 1 的向量。
*   **向量的单位化**：$\beta = \frac{\alpha}{|\alpha|}$（将非零向量转化为单位向量）。
*   **度量矩阵 (Metric Matrix/Gram Matrix)**：在任意基 $\{\alpha_i\}$ 下，内积可表示为 $(\alpha, \beta) = Y^T G X$，其中 $G = (g_{ij})_{n \times n} = (\alpha_i, \alpha_j)$。
    *   **结论**：基是标准正交基的充要条件是其度量矩阵 $G = E$（单位阵）。
### 3. 定理、推论与公式
*   **定理 2-1（Cauchy-Schwarz 不等式）**：
    $$|(\alpha, \beta)| \le |\alpha| |\beta|$$
    等号成立当且仅当 $\alpha, \beta$ 线性相关。
*   **推论（三角不等式）**：
    $$|\alpha + \beta| \le |\alpha| + |\beta|$$
*   **定理（勾股定理的抽象形式）**：若 $\alpha \perp \beta$，则 $|\alpha + \beta|^2 = |\alpha|^2 + |\beta|^2$。

### 4. 系统思维
*   **产生原因**：第一章的空间是“橡皮泥”式的，没有长短和角度。为了处理物理中的功、投影、最小误差等问题，必须引入度量。
*   **关联性**：内积 $\to$ 模（长度） $\to$ 距离。
*   **之后用途**：内积是所有优化算法（如梯度下降）、信号处理（正交分解）的几何基础。

## 2.2 正交基及子空间的正交关系

### 1. 核心定义
*   **正交向量组**：一组两两正交的非零向量。
*   **正交基 (Orthogonal Basis)**：由正交向量构成的基。
*   **标准正交基 (Orthonormal Basis)**：两两正交且模长均为 1 的基。满足 $(\epsilon_i, \epsilon_j) = \delta_{ij}$（克罗内克符号）。
*   **定义 2-4 (正交子空间, P34)**：设 $V_1, V_2$ 是 $V$ 的子空间，若对 $\forall \alpha \in V_1, \beta \in V_2$ 均有 $(\alpha, \beta) = 0$，则称 $V_1$ 与 $V_2$ **正交**，记为 $V_1 \perp V_2$。
*   **定义 2-5 (正交补, P35)**：设 $V_1$ 是 $V$ 的子空间，满足 $V_1 \perp V_1^\perp$ 且 $V = V_1 \oplus V_1^\perp$ 的子空间 $V_1^\perp$ 称为 $V_1$ 的**正交补**。

### 2. 定理、推论与算法
*   **性质**：正交向量组必定线性无关。
*   **定理 2-2（存在性）**：有限维内积空间必存在标准正交基。
*   **Schmidt 正交化过程 (Algorithm)**：将线性无关组 $\{\alpha_1, \dots, \alpha_n\}$ 转化为标准正交基 $\{\epsilon_1, \dots, \epsilon_n\}$。
    *   $\beta_1 = \alpha_1$
    *   $\beta_2 = \alpha_2 - \frac{(\alpha_2, \beta_1)}{(\beta_1, \beta_1)}\beta_1$
    *   $\beta_k = \alpha_k - \sum_{i=1}^{k-1} \frac{(\alpha_k, \beta_i)}{(\beta_i, \beta_i)}\beta_i$
    *   最后将 $\beta_i$ 单位化得到 $\epsilon_i$。
*   **定理 2-3 (正交直和, P34)**：若子空间 $V_1, V_2, \dots, V_k$ 两两正交，则它们的和必为直和。
*   **定理 2-4 (正交补存在性, P35)**：有限维内积空间的任何子空间 $V_1$ 都有唯一确定的正交补 $V_1^\perp$，且：
    $$\dim V_1 + \dim V_1^\perp = n$$
*   **推论 (P35)**：$(V_1^\perp)^\perp = V_1$。
*   **例题结论（重要）**：在标准正交基下，内积的计算简化为坐标的简单乘累加（类似标准 $\mathbb{R}^n$）。

### 3. 系统思维
*   **产生原因**：普通基下的内积计算非常麻烦（涉及度量矩阵）。
*   **关联性**：标准正交基就是线性空间里的“笛卡尔坐标系”。
*   **之后用途**：极大地简化了线性变换的矩阵表示，是矩阵对角化的前奏。

## 2.3 内积空间的同构

### 1. 核心定义
*   **内积空间同构 (Definition 2-6)**：设 $V, V'$ 为两欧氏空间。若存在线性同构映射 $\sigma: V \to V'$，且满足内积不变性：
    $$(\sigma\alpha, \sigma\beta) = (\alpha, \beta)$$
    则称其为内积空间同构（等距同构）。

### 2. 定理与推论
*   **定理 2-5**：所有 $n$ 维欧氏空间都同构于 $\mathbb{R}^n$（标准欧氏空间）。
*   **同构的性质**：同构映射保持向量的长度、夹角、正交性。

### 3. 系统思维
*   **产生原因**：证明了只要维数相同，所有欧氏空间的度量结构都是一致的。
*   **用途**：理论上我们可以只研究 $\mathbb{R}^n$，其结论可以直接推广到函数空间或矩阵空间。

## 2.4 正交变换

### 1. 核心定义
*   **正交变换 (Definition 2-7)**：内积空间 $V$ 上的线性变换 $T$，若保持内积不变：
    $$(T\alpha, T\beta) = (\alpha, \beta)$$
*   **正交矩阵 (Definition 2-11)**：满足 $A^T A = I$ 或 $A^{-1} = A^T$ 的实方阵。

### 2. 定理与推论（判定定理 2-6）
以下命题等价：
1. $T$ 是正交变换。
2. $T$ 保持向量模长不变：$|T\alpha| = |\alpha|$。
3. $T$ 将标准正交基映射为标准正交基。
4. $T$ 在标准正交基下的矩阵 $A$ 是**正交矩阵**。

### 3. 例题引申的新概念
*   **单位正交算子/变换**：即正交变换的另一种称呼。
*   **旋转变换与镜像变换**：是典型的正交变换。
*   **性质**：正交矩阵的行列式 $\det A = \pm 1$。
    *   $\det A = 1$ 称为第一类正交变换（旋转）。
    *   $\det A = -1$ 称为第二类正交变换（含镜像）。

### 4. 系统思维
*   **产生原因**：研究在不改变物体形状和大小的情况下，如何进行刚体运动。
*   **关联性**：正交变换 $\iff$ 保持几何特性的算子 $\iff$ 正交矩阵。
*   **之后用途**：在奇异值分解（SVD）和 QR 分解中，正交矩阵用于保持数值计算的稳定性（误差不放大）。

## 2.5 点到子空间的距离与最小二乘法

### 1. 核心定义
*   **定义 2-8（距离）**：向量 $\alpha, \beta$ 的距离定义为 $d(\alpha, \beta) = |\alpha - \beta|$。满足非负性、对称性、三角不等式。
*   **定义（正交投影）**：设 $W$ 是 $V$ 的子空间，对任意 $\alpha \in V$，若能分解为 $\alpha = \alpha_W + \alpha_{\perp}$，其中 $\alpha_W \in W$ 且 $\alpha_{\perp} \perp W$，称 $\alpha_W$ 为 $\alpha$ 在 $W$ 上的**正交投影**。
*   **最小二乘解**：对于矛盾方程组 $AX=B$（无解时），寻找一个 $X^*$ 使得误差向量的模 $|AX^* - B|$ 最小。

### 2. 定理、推论与公式
*   **距离最短定理**：点 $\alpha$ 到子空间 $W$ 内各点距离中，以到其正交投影 $\alpha_W$ 的距离最短。
*   **法方程 (Normal Equation)**：求 $AX=B$ 的最小二乘解等价于求解相容方程组：
    $$A^T AX = A^T B$$
*   **例题结论（重要）**：正交投影 $\alpha_W$ 是 $\alpha$ 在子空间 $W$ 中的**最佳逼近**。
*   **结论 (P39)**：若 $A$ 为列满秩，则最小二乘解唯一，且 $X = (A^T A)^{-1} A^T B$。

### 3. 系统思维
*   **产生原因**：现实中由于观测误差，方程组往往是矛盾的。最小二乘法提供了“退而求其次”的最优处理方案。
*   **关联性**：正交性 $\to$ 投影 $\to$ 误差最小。
*   **之后用途**：统计学中的回归分析、信号处理中的自适应滤波、机器学习中的损失函数优化。

## 2.6 复内积空间（酉空间）

### 1. 核心定义
*   **定义 2-9（复内积）**：设 $V$ 为复数域 $\mathbb{C}$ 上的线性空间，内积满足：
    1.  **共轭对称性**：$(\alpha, \beta) = \overline{(\beta, \alpha)}$
    2.  **齐次性**：$(k\alpha, \beta) = k(\alpha, \beta)$
    3.  **可加性**：$(\alpha + \beta, \gamma) = (\alpha, \gamma) + (\beta, \gamma)$
    4.  **正定性**：$(\alpha, \alpha) > 0$（当 $\alpha \neq 0$）
    称其为**酉空间 (Unitary Space)**。
*   **共轭转置 (Conjugate Transpose, $A^H$)**：$A^H = \overline{A}^T$。
*   **酉矩阵 (Unitary Matrix)**：满足 $A^H A = I$ 或 $A^{-1} = A^H$。
*   **酉变换**：保持复内积不变的线性变换。

### 2. 关键性质（例题与推论）
*   **注意点**：复内积对第二分量满足“共轭齐次性”：$(\alpha, k\beta) = \overline{k}(\alpha, \beta)$。
*   **复数域下的 Cauchy 不等式**：依然成立，但需加绝对值 $|(\alpha, \beta)| \le |\alpha| |\beta|$。
*   **酉变换判定**：与正交变换类似，酉变换保持模长不变，且矩阵为酉矩阵。

### 3. 系统思维
*   **产生原因**：实数域在代数上是不封闭的（如特征方程可能无实根）。为了保证所有算子都有特征值，必须扩展到复数域。
*   **关联性**：酉空间是欧氏空间在复数域的自然推广。
*   **之后用途**：量子力学（波函数在酉空间）、傅里叶变换（酉算子）。

## 2.7 正规矩阵

### 1. 核心定义
*   **定义 2-12（正规矩阵）**：满足 $AA^H = A^H A$ 的方阵 $A$ 称为**正规矩阵**。
*   **重要子类（例题结论）**：
    *   **厄米特矩阵 (Hermitian)**：$A^H = A$（特征值为实数）。
    *   **斜厄米特矩阵 (Skew-Hermitian)**：$A^H = -A$。
    *   **酉矩阵 (Unitary)**：$A^H = A^{-1}$（特征值模长为 1）。
    *   **实对称矩阵**、**实反对称矩阵**、**正交矩阵**。

### 2. 定理与核心结论
*   **Schur 定理**：任何方阵都酉相似于一个上三角矩阵。
*   **定理 2-8（谱定理）**：矩阵 $A$ 酉相似于对角阵的充要条件是 **$A$ 是正规矩阵**。
    $$Q^H AQ = \text{diag}(\lambda_1, \dots, \lambda_n)$$

### 3. 系统思维
*   **产生原因**：我们想知道什么样的矩阵可以“最完美地降维”（即对角化）。正规矩阵就是这个问题的终极答案。
*   **关联性**：正规矩阵涵盖了几乎所有物理和工程中常见的优良矩阵。
*   **之后用途**：这是第三章矩阵标准形理论的巅峰。

## 2.8 厄米特二次型

### 1. 核心定义
*   **定义 2-13（厄米特二次型）**：$f(X) = X^H AX$，其中 $A$ 为厄米特矩阵。
*   **性质**：厄米特二次型的值对于任何 $X$ 恒为**实数**。
*   **定义 2-14（正定性）**：若 $X \neq 0$ 时 $f(X) > 0$，称二次型及 $A$ 是**正定的**。

### 2. 定理与判定
*   **定理 2-10**：厄米特二次型总可以通过酉变换化为标准形（平方和形式）。
*   **定理 2-11（正定充要条件）**：$A$ 的特征值全部大于 0，或各阶顺序主子式全部大于 0。

## 2.9 力学系统的微小振动

### 1. 知识点总结（物理应用）
*   **能量表达**：
    *   动能 $T = \frac{1}{2}\dot{q}^T M \dot{q}$（$M$ 为质量阵，正定）。
    *   势能 $U = \frac{1}{2}q^T K q$（$K$ 为刚度阵）。
*   **振动方程**：$M\ddot{q} + Kq = 0$。
*   **重要结论**：利用两个二次型（$M$ 和 $K$）的**同时对角化**，可以将耦合的振动系统分解为独立的“模态”。
*   **广义特征值问题**：$\det(K - \lambda M) = 0$。

### 2. 系统思维
*   **产生原因**：处理多自由度系统的复杂振动。
*   **关联性**：线性空间（坐标）+ 内积空间（能量度量）+ 矩阵相似理论。
*   **用途**：桥梁、建筑、航天器的模态分析。
