# Obsidian 已安装插件功能指南

## 1. Calendar (日历)
*   **功能**：在右侧边栏显示一个直观的月历视图，专门用于管理**日记 (Daily Notes)**。
*   **核心用法**：
    *   **点击日期**：直接创建或跳转到当天的日记。
    *   **字数统计**：日期下方会出现圆点，圆点越多代表当天写的字数越多（可视化你的写作习惯）。
    *   **周记支持**：如果在设置里开启，点击周码（如 W01）可以创建周记。

## 2. Commander (命令官)
*   **功能**：允许你将任何 Obsidian 的命令（Command Palette 里的命令）以**图标**的形式添加到界面的各个位置。
*   **核心用法**：
    *   **添加位置**：可以将按钮加到页面标题栏、状态栏、右键菜单、甚至移动端的工具栏。
    *   **宏 (Macro)**：可以将多个命令组合成一个按钮，一键执行一套动作。
    *   **设置**：进入插件设置，选择要添加图标的区域（例如 Tab bar），搜索命令并选择图标即可。

## 3. Dataview (数据视图) - *神器*
*   **功能**：Obsidian 最强大的插件之一，把你的笔记库变成一个数据库。你可以通过编写查询语句来自动汇总笔记。
*   **核心用法**：
    *   **基本查询**：使用代码块 \`\`\`dataview\`\`\`。
    *   **示例**：列出所有标签为 #book 的笔记：
        ```dataview
        LIST FROM #book
        ```
    *   **表格视图**：`TABLE file.ctime as "创建时间" FROM "文件夹名"`。

## 4. Editor Width Slider (编辑器宽度滑块)
*   **功能**：允许你自由调节编辑器文本内容的显示宽度，而不需要修改 CSS 代码。
*   **核心用法**：
    *   **启用**：安装后，状态栏（右下角）会出现一个滑块或者百分比数字。
    *   **调节**：拖动滑块即可改变正文的宽窄，适合在大屏幕上居中专注写作。

## 5. Excalidraw (画板)
*   **功能**：在 Obsidian 中集成全功能的虚拟白板，支持手绘、流程图、思维导图，并且可以双向链接到笔记。
*   **核心用法**：
    *   **创建**：点击新建 Excalidraw 文件。
    *   **链接**：在画板中按 `[` 可以引用笔记；在笔记中可以嵌入画板。
    *   **库**：支持导入素材库（图标、图表模板）。

## 6. Git
*   **功能**：利用 Git 版本控制系统对笔记库进行备份。支持自动推送到 GitHub/GitLab 等远程仓库。
*   **核心用法**：
    *   **前置条件**：电脑上需要先安装 Git 软件。
    *   **自动备份**：在设置中配置 `Backup interval`（备份间隔），插件会自动执行 add, commit 和 push。
    *   **版本回退**：可以通过 Git 历史找回误删或修改前的内容。

## 7. Heatmap Calendar (热力图日历)
*   **功能**：生成类似 GitHub 贡献图的“热力图”，常用于习惯打卡或追踪某些数据的频率。
*   **核心用法**：
    *   通常配合 DataviewJS 使用。
    *   需要使用 \`\`\`heatmap_calendar\`\`\` 代码块，并输入符合 JSON 格式的数据来渲染颜色深浅。

## 8. Image Toolkit (图片工具箱)
*   **功能**：增强 Obsidian 的图片查看体验。
*   **核心用法**：
    *   **查看**：点击笔记中的图片，会以灯箱模式（弹窗）打开。
    *   **操作**：支持滚轮缩放、旋转、镜像翻转、甚至反色（看黑底白字的图很方便）。

## 9. Kanban (看板)
*   **功能**：在 Obsidian 中实现类似 Trello 的看板管理，非常适合做项目管理或任务流转。
*   **核心用法**：
    *   **创建**：新建一个看板文件。
    *   **列表**：创建“未开始”、“进行中”、“已完成”等列。
    *   **卡片**：在列中添加卡片，卡片可以是纯文字，也可以是一个笔记的链接。可以在设置中关联 Markdown 的复选框任务。

## 10. Latex Suite (Latex 套件)
*   **功能**：极大提高数学公式（LaTeX）的输入速度，主要通过“片段展开”实现。
*   **核心用法**：
    *   **快捷输入**：例如设置输入 `mk` 自动展开为 `$ $`（行内公式），输入 `/` 自动变成 `\frac{}{}`（分数）。
    *   **适用人群**：理工科学生或需要大量输入数学符号的用户。

## 11. Omnisearch (全能搜索)
*   **功能**：比 Obsidian 自带搜索更强大的搜索引擎。
*   **核心用法**：
    *   **OCR**：支持搜索图片和 PDF 中的文字（需要在设置里下载对应的语言模型）。
    *   **模糊搜索**：即使有轻微的拼写错误也能搜到内容。
    *   **使用**：通常用它替换默认的 `Ctrl/Cmd + F` 或 `Ctrl/Cmd + O` 搜索界面。

## 12. Paste image rename (粘贴图片重命名)
*   **功能**：解决了 Obsidian 粘贴图片默认为 "Pasted image 2023..." 这种无意义文件名的问题。
*   **核心用法**：
    *   **自动触发**：当你截图并粘贴到笔记时，插件会自动弹出一个输入框。
    *   **重命名**：输入你想好的名字，回车，图片就会以新名字保存并链接到当前位置。

## 13. QuickAdd (快速添加)
*   **功能**：自动化工作流插件，用于快速捕捉想法、插入模板或执行宏命令。
*   **核心用法**：
    *   **Capture (捕捉)**：不打开笔记的情况下，快速追加一行字到指定笔记（如“今日闪念”）。
    *   **Template (模板)**：比自带模板更强，支持重命名文件、移动文件。
    *   **Macro (宏)**：可以将多个动作（如：创建文件 -> 移动 -> 插入模板 -> 打开图谱）串联起来。

## 14. Style Settings (样式设置)
*   **功能**：为第三方主题提供图形化的设置界面。
*   **核心用法**：
    *   **依赖**：你的当前主题（Theme）必须支持 Style Settings。
    *   **调节**：在设置里，你可以像调参数一样修改字体大小、颜色、侧边栏宽度、标题样式等，而不需要懂 CSS 代码。

## 15. Templater (模板增强)
*   **功能**：比 Obsidian 自带的核心插件“模板”强大得多的模板引擎，支持 JavaScript。
*   **核心用法**：
    *   **动态变量**：使用 `<% tp.date.now() %>` 插入当前时间，`<% tp.file.title %>` 插入文件名。
    *   **光标跳转**：设置 `<% tp.file.cursor() %>`，插入模板后光标自动跳到该位置，方便直接开始打字。
    *   **脚本执行**：可以调用系统命令或运行复杂的 JS 脚本来生成内容。

---

### 💡 插件联动小贴士

1.  **QuickAdd + Templater**: 使用 QuickAdd 触发创建笔记的动作，然后利用 Templater 自动填充复杂的日期、格式和元数据。
2.  **Dataview + Kanban**: 在看板中管理项目，用 Dataview 在每日笔记中自动汇总“进行中”的任务。
3.  **Excalidraw + Image Toolkit**: 虽然 Excalidraw 主要用于画图，但配合 Image Toolkit 可以更好地浏览你收集的图片素材。
